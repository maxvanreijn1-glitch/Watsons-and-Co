# Watsons & Co - Luxury Estate Agents Website

A premium, full-scale website for **Watsons & Co**, a luxury estate agents company. Built with a sophisticated white and black color scheme, this website showcases high-end residential and commercial properties.

## 🏠 Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Luxury Aesthetic**: White primary color with black secondary elements and gold accents
- **Property Showcase**: Featured properties section with detailed cards
- **Advanced Search**: Property filtering by location, type, and price range
- **Services Overview**: Comprehensive listing of real estate services
- **Client Testimonials**: Social proof from satisfied clients
- **Contact Section**: Email form and full contact information
- **Smooth Animations**: Intersection observer-based fade-in effects
- **Modern Navigation**: Sticky navbar with smooth scrolling
- **Newsletter Signup**: Email capture for marketing
- **SEO Optimized**: Semantic HTML structure
